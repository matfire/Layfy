# Layfy

Control Spotify from a browser popup

## Features

- OAuth connect with spotify
- Play / Resume
- Skip forward and backward
- Set Repeat mode (off, track, context)
- Mute / Unmute
- "Search on Spotify" context menu
- Custom Keybindings for easy control
